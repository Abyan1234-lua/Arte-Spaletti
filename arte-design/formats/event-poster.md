# FORMAT: Event Poster

Use this template for: conferences, concerts, brand launches, networking events,  
community gatherings, festivals, exhibitions, product launch parties.

---

## PROMPT TEMPLATE

```
Professional event promotional poster, {{aspect_ratio}}, high-impact visual design.

BACKGROUND: {{background_type}} — {{bg_description}}. Color palette:
{{bg_color_primary}} as dominant with {{bg_color_secondary}} accent zones.
{{bg_effect}} layered subtly for atmosphere. Overall mood: {{mood}}.

VISUAL HERO: {{hero_element}} as the dominant visual element, positioned
{{hero_position}}, occupying approximately {{hero_size}}% of the canvas.
{{hero_style}} treatment. {{hero_detail}}.

TOP ZONE: Upper portion of poster — flat {{top_zone_color}} area or blended
with background, clean open space for organizer identity. May include simple
logo-like placeholder or remain clean for logo replacement. Approximately
top 12% of canvas height.

TITLE ZONE: Center to upper-center of poster — bold flat open area, visually
framed by {{frame_element}}, large space for event name in dominant typography.
Zone approximately 60% canvas width, 25% canvas height. Keep clean and readable
for easy text refinement.

DATE/LOCATION STRIP: Center-lower section — {{date_strip_color}} band,
semi-transparent or solid, spanning 80% canvas width, divided into date area
(left) and location area (right) by a thin {{divider_color}} vertical line.
Text may be generated but must remain clean and replaceable.

LINEUP / SPEAKERS ZONE: Lower-center — structured section with {{lineup_cols}}
portrait placeholders (circles or rectangles) arranged in a row, each with
name text or placeholder below. Optional depending on event type.

CTA / TICKET ZONE: Bottom — {{cta_color}} bar or pill shape, centered,
prominent call-to-action area. May include generated CTA text, but must remain
clear and editable.

LOGO AREA: Top or header area — may include placeholder mark or simple identity,
intended for replacement with real logo in Canva.

QR CODE AREA (optional): Small square zone if needed — generate as visual
placeholder only.

TEXT HANDLING: If no text content is provided, generate realistic event text
(event name, tagline, date, location, CTA, lineup names if applicable)
appropriate to the theme. All text must be clean, readable, and easily
replaceable.

COLOR PALETTE: {{primary_color}}, {{secondary_color}}, {{accent_color}}.

STYLE: {{style_directive}}, high production value event design, commercial
print quality, dramatic visual impact, {{lighting_style}}, balanced composition,
editable-friendly structure, no watermark.

ASPECT RATIO: {{aspect_ratio}}
```

## CANVA EDITING ZONES (UPDATED FOR MAGIC LAYERS)


| Zone               | Location              | What to do in Canva                   |
| ------------------ | --------------------- | ------------------------------------- |
| Organizer logo     | Top center / top-left | Replace placeholder with real logo    |
| Event name         | Title zone            | Refine or replace main title          |
| Event tagline      | Below title           | Adjust messaging                      |
| Date               | Date strip left       | Replace with actual date              |
| Location           | Date strip right      | Update venue + city                   |
| Lineup             | Portrait row          | Replace with real speakers/performers |
| Ticket CTA         | Bottom CTA zone       | Replace text + add link               |
| Hashtag            | Bottom corner         | Add or refine hashtag                 |
| QR code (optional) | QR zone               | Replace with real QR                  |
| Cleanup            | Entire layout         | Remove artifacts, fix alignment       |
