# FORMAT: Webinar / Live Seminar Banner

Use this template for: online webinars, Zoom seminars, live sessions, virtual  
conferences, online workshops, LinkedIn Live events.

Reference style: Zoom-style seminar poster — clean corporate layout,  
speaker photo prominent, structured info blocks, strong CTA.

---

## PROMPT TEMPLATE

```
Professional webinar event banner, {{aspect_ratio}}, digital and print-ready.

BACKGROUND: Clean {{background_style}} background — {{bg_color_primary}} upper
section blending into {{bg_color_secondary}} lower section. Subtle {{bg_texture}}
geometric shape or soft gradient blob on {{blob_position}} side, color
{{blob_color}}, low opacity decorative element only. Overall feel: clean,
professional, trustworthy.

HEADER STRIP: Top-left area — clean flat zone, {{header_strip_color}} or white,
reserved for logo and event type label. Approximately 15% of canvas height.
May include simple placeholder mark or generated label, but must remain clean
and replaceable.

SPEAKER PHOTO ZONE: {{photo_position}} — large portrait area, person looking
{{gaze_direction}}, professional attire, {{photo_bg_treatment}} background
blending smoothly into poster background. Photo occupies approximately 45% of
canvas width. Soft vignette edges, realistic professional headshot quality.

TITLE ZONE: {{title_position}} — large clean area, {{title_bg}} background,
space for bold multi-line event title. May include generated title text but
must remain highly readable and easy to refine.

DESCRIPTION AREA: Below title — smaller clean strip for 2–3 lines of event
description. Text may be generated but must remain concise and editable.

SPEAKER INFO ZONE: Below title/description — structured strip with thin
{{accent_color}} left border line, space for speaker name and title.
Flat {{speaker_strip_color}} background. Text may be generated but must remain
clean and replaceable.

INFO BLOCK STRIP: Lower section full-width — {{info_strip_color}} band,
divided into {{info_columns}} equal columns by subtle divider lines.
Each column contains icon + short text (may be generated) representing:
date, time, platform, audience.

CTA STRIP: Bottom full-width — {{cta_bg_color}} band.
Left: short urgency text area (editable)
Center: {{cta_button_color}} pill/rounded button (CTA text may be generated)
Right: small square QR placeholder (visual only)

LOGO AREA: Header or top corner — may include placeholder mark or clean space
for real logo replacement.

QR CODE AREA: Bottom-right — always a visual placeholder only.

TEXT HANDLING: If no text is provided, generate realistic webinar content
(title, speaker name, description, date, time, CTA). All text must be readable,
structured, and easily replaceable.

COLOR PALETTE: Primary {{primary_color}}, accent {{accent_color}},
info strip {{info_strip_color}}, CTA {{cta_bg_color}}.

STYLE: Clean corporate design, professional, flat design for text zones,
photorealistic speaker portrait, smooth gradients, modern layout,
balanced whitespace, editable-friendly structure, no watermark.

ASPECT RATIO: {{aspect_ratio}}
```

## CANVA EDITING ZONES (UPDATED FOR MAGIC LAYERS)


| Zone             | Location           | What to do in Canva                |
| ---------------- | ------------------ | ---------------------------------- |
| Logo             | Header strip       | Replace placeholder with real logo |
| Event type label | Header             | Adjust label text                  |
| Badge            | Top corner         | Add/remove tag (FREE, PAID, etc.)  |
| Main title       | Title zone         | Refine headline                    |
| Description      | Below title        | Adjust short description           |
| Speaker name     | Info zone          | Replace/refine name                |
| Speaker title    | Below name         | Edit credentials                   |
| Company logo     | Below speaker info | Replace placeholder                |
| Date             | Info column        | Update                             |
| Time             | Info column        | Update                             |
| Platform         | Info column        | Update                             |
| Audience         | Info column        | Adjust targeting                   |
| CTA text         | Bottom left        | Refine urgency                     |
| CTA button       | Center             | Replace text + link                |
| QR code          | Bottom right       | Replace with real QR               |
| Cleanup          | Entire layout      | Fix spacing, alignment, artifacts  |
