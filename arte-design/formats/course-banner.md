# FORMAT: Course / E-Learning Banner

Use this template for: online course announcements, e-learning platform banners,  
training program promotions, bootcamp ads, masterclass promotions,  
certification program banners.

---

## PROMPT TEMPLATE

```
Professional online course promotional banner, {{aspect_ratio}},
educational marketing design.

BACKGROUND: {{background_style}} — {{bg_description}}.
Primary color: {{bg_color_primary}}. Secondary zone: {{bg_color_secondary}}.
{{bg_geometric}} clean geometric shapes or subtle abstract elements in
{{geometric_color}} at {{geometric_position}}, low opacity. Professional and
trustworthy feel.

INSTRUCTOR PHOTO ZONE: {{photo_position}} — professional portrait of instructor,
{{photo_style}} attire, confident and approachable expression, seamlessly
integrated into background via {{photo_treatment}}. Photo approximately
{{photo_size}}% of canvas. If no instructor provided, generate a neutral
professional placeholder subject or abstract substitute.

COURSE VISUAL METAPHOR: {{visual_metaphor}} element positioned at
{{metaphor_position}}, representing the course subject matter. {{metaphor_style}}
treatment. Subtle, supporting role — not dominant.

COURSE TYPE BADGE: Small flat {{badge_color}} pill or tag shape at top or near
headline zone. May include simple readable label or placeholder text.

HEADLINE ZONE: {{headline_position}} — large flat open area, {{headline_bg}}
clean background, space for course title in bold typography, 2–3 lines.
Approximately 50% canvas width, 35% canvas height. Keep clean for easy text
refinement.

OUTCOMES ZONE: Below headline — structured panel with {{outcomes_count}}
horizontal rows, each with a small {{check_icon_color}} icon or placeholder
on left, and text bar or simple text on right representing learning outcomes.

STATS STRIP: Horizontal band, {{stats_color}} fill, divided into
{{stats_count}} equal sections. Each section includes icon + number/text
placeholder representing students, rating, duration, lessons.

CTA ZONE: Bottom — {{cta_bg_color}} band. Left section: urgency text area.
Center: prominent {{cta_button_color}} rounded button. Right: price display
area with {{price_styling}} treatment. All elements editable.

LOGO AREA: Optional top corner or header area — may include a simple
placeholder mark or remain clean for logo replacement.

QR CODE AREA (optional): Small square zone if needed — generate as visual
placeholder only.

TEXT HANDLING: If no text content is provided by the user, generate complete,
realistic course text (title, outcomes, stats, CTA, pricing) appropriate to
the theme. All generated text must remain clean, readable, and easily
replaceable.

COLOR PALETTE: {{primary_color}}, {{secondary_color}}, {{accent_color}}.

STYLE: Professional e-learning marketing aesthetic, trustworthy and credible,
{{industry_style}} feel, clean structured layout, {{lighting_style}},
high clarity, balanced composition, editable-friendly structure.

ASPECT RATIO: {{aspect_ratio}}
```

## CANVA EDITING ZONES (UPDATED FOR MAGIC LAYERS)


| Zone               | Location          | What to do in Canva                |
| ------------------ | ----------------- | ---------------------------------- |
| Platform logo      | Top-left / header | Replace placeholder with real logo |
| Course type        | Badge zone        | Refine or replace text             |
| Course title       | Headline zone     | Replace AI text with final title   |
| Outcomes           | Outcome rows      | Edit or rewrite learning points    |
| Students count     | Stats section     | Replace with real data             |
| Rating             | Stats section     | Adjust value and format            |
| Duration           | Stats section     | Update course length               |
| Lessons            | Stats section     | Update lesson count                |
| Urgency text       | CTA left          | Refine messaging                   |
| Enroll button      | CTA center        | Replace text + add link            |
| Price              | CTA right         | Adjust pricing                     |
| Instructor name    | Near photo        | Add real name + credentials        |
| QR code (optional) | QR zone           | Replace with real generated QR     |
| Cleanup            | Entire layout     | Remove artifacts, fix alignment    |
