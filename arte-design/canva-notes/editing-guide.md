# CANVA EDITING GUIDE — Universal Rules

This guide applies to ALL ARTE-DESIGN outputs. Every GPT Image prompt is  
engineered with these Canva editing rules in mind.

---

## WHY THIS EXISTS

GPT Image generates the **full visual composition (including all assets, layout, and elements)**.  
Canva handles all generated images using its feature called **Magic Layers**.

This two-step workflow means:

---

GPT Image = Complete Generated Design

Canva = Editing, separation, and refinement via Magic Layers

---

## GOLDEN RULES FOR ALL FORMATS

### Rule 1 — Design Must Be Cleanly Separable

Every prompt should create layouts that are easy to edit using Magic Layers.  
This means:

- Clear spacing between elements
- Avoid excessive overlap
- Maintain readable structure and grouping

---

### Rule 2 — Text Is Not Final

Text may appear in generated images, but it is:

- Often inaccurate or distorted
- Not true editable typography

Always remove, refine, or replace text inside Canva using proper text elements.

---

### Rule 3 — Logos Must Be Replaced

Generated logos or brand marks are not reliable.

- They may look correct but are not accurate
- They are not usable for production

Always upload your official logo (PNG transparent) and replace it in Canva.

---

### Rule 4 — QR Codes Must Be Replaced

Never rely on generated QR codes — they are invalid.

Generate real QR codes using:

- [qr.io](http://qr.io)
- [qrcode-monkey.com](http://qrcode-monkey.com)
- canva.com/qr-code-generator

Then replace the generated one inside Canva.

---

## CANVA WORKFLOW (Step-by-Step)

1. Generate image in GPT Image using ARTE-DESIGN prompt
2. Download the generated image (PNG or JPG)
3. Open Canva → Create new design → Set exact dimensions
4. Upload generated image → Set as background (fill frame)
5. Use Magic Layers to:
  - Remove unwanted elements
  - Adjust composition
  - Clean up areas for text
6. Replace or refine text using Canva text tools
7. Upload logo PNG (transparent bg) → Replace generated logo
8. Add QR code if needed → Generate externally → Upload → Replace
9. Add icons from Canva icon library if needed
10. Final check: clean artifacts, fix alignment, verify readability
11. Export: PNG for digital, PDF Print for print formats

## TEXT SIZE HIERARCHY

Use this scale relative to canvas size:


| Text Type       | Relative Size       | Canva Example (1080px) |
| --------------- | ------------------- | ---------------------- |
| Main headline   | 12–16% canvas width | 120–160pt              |
| Sub-headline    | 6–8% canvas width   | 60–80pt                |
| Body copy       | 3–4% canvas width   | 30–40pt                |
| Label / Caption | 2–2.5% canvas width | 20–24pt                |
| Fine print      | 1.5% canvas width   | 14–16pt                |

## COMMON CANVA MISTAKES TO AVOID

❌ Trusting AI-generated text without editing  
❌ Over-editing and breaking layout structure  
❌ Using low contrast text — always verify readability  
❌ Stretching logo — maintain aspect ratio  
❌ Keeping fake QR codes  
❌ Not cleaning artifacts after Magic Layer edits  
❌ Not aligning to grid — use Canva rulers and guides  
❌ Carousel font inconsistency — apply same font across slides

## CAROUSEL-SPECIFIC CANVA RULES

- Create all slides as separate pages within one Canva file
- Use "Copy page" after completing slide 1 — paste for all subsequent slides
- Lock the background image layer on each slide to prevent accidental moves
- Use Canva’s Brand Kit to lock your palette and fonts across all slides
- Keep left margin consistent using Canva rulers
- Download carousel as individual PNGs → upload as multiple images
