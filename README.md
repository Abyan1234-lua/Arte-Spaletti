Arte-Spaletti is a prompt engineering framework in form of Claude Skill for graphic designers using Generative AI on image generation (Nano Banana, GPT Image 2.0) to produce efficient prompt without doing wall of text prompt typing. It focuses on turning visual intent into structured prompts that produce consistent, high-quality results.

Inspiration

Arte-Spaletti is inspired by Name of Anime Character "Arte Spaletti" from Arte, a story about discipline, craftsmanship, and growth as an artist. That same spirit is applied here: mastering prompts as a creative craft.

Current Added Skill
1. arte-flow : Designed for creating high-quality artwork for entertainment purposes.
